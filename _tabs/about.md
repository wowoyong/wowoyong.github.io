# AI 데일리 소개

**AI 데일리**는 매일 AI 관련 뉴스와 주목할 GitHub 프로젝트를 한국어로 요약 정리하는 블로그입니다.

## 다루는 내용

- **AI 뉴스** — LLM 업데이트, 생성형 AI 신제품, 연구 논문 요약
- **GitHub 하이라이트** — 주목할 오픈소스 AI 프로젝트, 트렌딩 레포지토리
- **주간 정리** — 한 주간의 주요 흐름 요약
