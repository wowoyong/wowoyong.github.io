## 오늘의 AI GitHub 프로젝트

### 1. [microsoft/markitdown](https://github.com/microsoft/markitdown)

PDF, Word, PowerPoint 등 다양한 문서 형식을 마크다운으로 변환하는 Python 라이브러리.

- ⭐ 40k+ stars
- LLM 파이프라인에 문서를 넣을 때 전처리용으로 인기
- `pip install markitdown` 으로 바로 사용 가능

```python
from markitdown import MarkItDown
md = MarkItDown()
result = md.convert("document.pdf")
print(result.text_content)
```
