## 핵심 요약

학부 컴퓨터 네트워크는 컴퓨터 시스템들이 어떻게 정보를 주고받고 서로 소통하는지에 대한 핵심 원리를 다룹니다. 이 분야는 데이터를 전송하는 물리적 방법부터 애플리케이션이 정보를 활용하는 논리적 방식까지 광범위한 지식을 제공합니다. OSI 7 계층 및 TCP/IP 4 계층 모델을 통해 네트워크 통신의 복잡한 과정을 체계적으로 이해하고, 다양한 프로토콜과 장비의 역할을 학습합니다. 안정적이고 효율적인 통신 환경 구축뿐만 아니라, 네트워크를 외부 위협으로부터 보호하는 보안 기술 또한 중요하게 다뤄집니다. 현대 사회의 모든 정보 시스템과 서비스를 지탱하는 기반 기술로서, 컴퓨터 네트워크에 대한 이해는 미래 기술 전문가에게 필수적입니다.

## 네트워크란 무엇인가?

- 컴퓨터 및 장치들이 정보를 교환하기 위해 연결된 시스템입니다.
- 데이터, 자원, 서비스를 공유하며 효율성을 높이는 역할을 합니다.
- 유선(이더넷) 및 무선(와이파이) 연결 방식을 주로 사용합니다.
- 개인용 근거리 통신망(LAN)부터 전 세계적 인터넷까지 다양한 크기가 존재합니다.
- 정보 통신 기술 및 현대 사회의 모든 서비스의 기반 기술입니다.
- 물리적 연결 매체와 논리적 통신 규칙(프로토콜)의 조합으로 작동합니다.
- 인터넷은 가장 대표적인 전 세계적인 네트워크 사례입니다.
- 네트워크 성능은 통신 속도, 안정성, 보안 등에 직접적인 영향을 미칩니다.

## OSI 7 계층 모델

- 네트워크 통신 과정을 7단계로 표준화한 이론적 참조 모델입니다.
- 각 계층은 독립적인 기능을 수행하며 하위 계층에 서비스를 제공합니다.
- 물리 계층(1계층): 비트 전송, 물리적 매체(예: 케이블, 허브)를 담당합니다.
- 데이터 링크 계층(2계층): 프레임 전송, 오류 제어, MAC 주소를 사용하여 통신합니다. (예: 스위치)
- 네트워크 계층(3계층): 패킷 라우팅, IP 주소를 기반으로 경로를 설정합니다. (예: 라우터)
- 전송 계층(4계층): 종단 간 데이터 전송, 흐름/오류 제어, 포트 번호를 사용합니다. (예: TCP, UDP)
- 세션 계층(5계층): 통신 세션을 수립, 관리, 종료하는 기능을 담당합니다.
- 표현 계층(6계층): 데이터 암호화, 압축, 형식 변환 등 표현 방식을 담당합니다.
- 응용 계층(7계층): 사용자 인터페이스 및 네트워크 서비스(예: HTTP, FTP)를 제공합니다.

## TCP/IP 4 계층 모델

- 실제 인터넷에서 널리 사용되는 프로토콜 스택 모델입니다.
- OSI 7 계층 모델보다 간소화된 4단계로 구성되어 있습니다.
- 네트워크 접근 계층: 물리 계층과 데이터 링크 계층의 기능을 통합하며 MAC 주소를 사용합니다.
- 인터넷 계층: IP(인터넷 프로토콜)를 이용한 패킷 전송 및 라우팅을 담당합니다.
- 전송 계층: TCP(전송 제어 프로토콜)와 UDP(사용자 데이터그램 프로토콜)를 이용한 데이터 전송을 수행합니다.
- 응용 계층: HTTP, FTP, SMTP 등 다양한 응용 프로토콜이 포함됩니다.
- 실용성과 효율성으로 인해 현재 대부분의 네트워크 환경에서 사용됩니다.
- OSI 모델이 이론적이라면, TCP/IP 모델은 구현 중심의 현실적인 모델입니다.
- 각 계층은 특정 프로토콜과 장비들을 사용하여 데이터를 처리하고 전달합니다.
- 캡슐화를 통해 상위 계층의 데이터가 하위 계층에 포함되어 전송됩니다.

## 네트워크 프로토콜

- 네트워크에 연결된 장치 간 통신 규칙과 절차를 정의한 약속입니다.
- 데이터 교환의 일관성과 호환성을 보장하여 원활한 통신을 가능하게 합니다.
- IP(인터넷 프로토콜): 데이터 패킷의 주소 지정 및 라우팅을 담당하는 핵심 프로토콜입니다.
- TCP(전송 제어 프로토콜): 신뢰성 있는 연결 기반 데이터 전송을 제공하며, 오류 제어 및 흐름 제어를 수행합니다.
- UDP(사용자 데이터그램 프로토콜): 비연결형, 빠른 데이터 전송에 적합하며 실시간 스트리밍 등에 사용됩니다.
- HTTP(하이퍼텍스트 전송 프로토콜): 웹 브라우저와 웹 서버 간의 통신을 위한 프로토콜입니다.
- DNS(도메인 이름 시스템): 사람이 읽기 쉬운 도메인 이름을 IP 주소로 변환하여 연결을 돕습니다.
- ARP(주소 결정 프로토콜): IP 주소를 물리적 MAC 주소로 변환하여 로컬 네트워크 통신을 가능하게 합니다.
- ICMP(인터넷 제어 메시지 프로토콜): 네트워크 오류 메시지 전송 및 진단 기능을 제공합니다.
- 다양한 프로토콜이 각 계층에서 특정 역할을 수행하며 유기적으로 결합되어 네트워크를 구성합니다.

## 네트워크 장비

- 네트워크를 구성하고 데이터를 전송/처리하는 데 사용되는 하드웨어입니다.
- 허브: 여러 장치를 연결하고 모든 포트에 데이터를 전송하는 단순한 장치입니다.
- 스위치: MAC 주소를 기반으로 특정 포트에만 데이터를 전송하는 지능형 장치입니다. (데이터 링크 계층)
- 라우터: IP 주소를 기반으로 서로 다른 네트워크 간에 데이터 패킷의 경로를 지정합니다. (네트워크 계층)
- 게이트웨이: 서로 다른 프로토콜을 사용하는 네트워크를 연결하여 통신을 가능하게 합니다.
- 방화벽: 네트워크 보안을 위해 접근을 제어하고 유해 트래픽을 필터링합니다.
- 무선 접속 장치(AP): 무선 장치들이 유선 네트워크에 접속할 수 있도록 연결합니다.
- 모뎀: 디지털 신호를 아날로그 신호로, 또는 그 반대로 변환하여 통신을 가능하게 합니다.
- 네트워크 인터페이스 카드(NIC): 컴퓨터가 네트워크에 연결될 수 있도록 하는 하드웨어 부품입니다.

## 네트워크 보안 기초

- 네트워크 자원과 데이터를 무단 접근, 손상, 오용으로부터 보호하는 일련의 기술과 정책입니다.
- 기밀성: 인가된 사용자만이 정보에 접근하고 내용을 확인할 수 있도록 보장합니다.
- 무결성: 정보가 무단으로 변경되거나 파괴되지 않았음을 보장하여 신뢰성을 유지합니다.
- 가용성: 인가된 사용자가 필요할 때 정보 및 시스템에 접근할 수 있도록 보장합니다.
- 방화벽: 네트워크의 출입을 감시하고 제어하여 외부 침입을 막는 문지기 역할을 합니다.
- VPN(가상 사설망): 공중망을 통해 암호화된 터널을 만들어 안전한 사설 네트워크 연결을 제공합니다.
- 암호화: 데이터를 전송하거나 저장할 때 내용을 알아보지 못하게 변환하여 기밀성을 확보합니다.
- 인증: 사용자나 장치의 신원을 확인하여 접근 권한을 부여하는 절차입니다.
- 접근 제어: 인가된 사용자에게만 특정 자원에 대한 접근 권한을 부여하여 보안을 강화합니다.
- 정기적인 보안 업데이트, 취약점 점검 및 보안 정책 수립이 네트워크 보안에 중요합니다.

## 최신 흐름

- **클라우드 네트워킹**: 온프레미스 인프라 대신 클라우드 서비스 공급자를 통해 네트워크 자원을 구성하고 관리하는 추세가 확대되고 있습니다.
- **5G 및 미래 무선 통신**: 초고속, 초저지연, 대규모 연결성을 제공하여 사물 인터넷, 자율주행 등 혁신적인 서비스 발전을 가속화하고 있습니다.
- **엣지 컴퓨팅**: 데이터를 생성하는 근원지에 가까운 곳에서 데이터를 처리하여 지연 시간을 줄이고 중앙 서버의 부하를 분산하는 기술이 중요해지고 있습니다.
- **SDN**: 네트워크 제어 기능을 하드웨어에서 분리하여 소프트웨어로 관리함으로써 네트워크의 유연성, 자동화 및 프로그래밍 가능성을 극대화합니다.
- **NFV**: 라우터, 방화벽과 같은 기존의 하드웨어 기반 네트워크 기능을 범용 서버에서 소프트웨어 형태로 구현하여 비용 효율성과 유연성을 높입니다.
- **제로 트러스트 보안 모델**: '절대 신뢰하지 않고 항상 검증하라'는 원칙 아래, 내부/외부 사용자 및 장치의 모든 접근 시도를 엄격히 확인하고 권한을 최소화하는 접근 방식입니다.
- **사물 인터넷 네트워크**: 스마트 홈, 스마트 팩토리, 스마트 시티 등 수많은 장치가 서로 연결되어 데이터를 주고받으며 지능형 환경을 구축하는 기술이 더욱 발전하고 있습니다.
- **블록체인 기반 분산 네트워크**: 중앙 집중식 제어에서 벗어나 데이터 무결성과 보안을 강화하며, 분산원장기술을 활용한 새로운 네트워크 모델이 연구되고 있습니다.
- **AI 기반 네트워크 관리**: 인공지능과 머신러닝이 네트워크 트래픽 분석, 이상 감지, 성능 최적화, 자동 복구 등을 수행하여 네트워크 운영의 효율성을 혁신하고 있습니다.
- **양자 네트워크(초기 단계)**: 양자 얽힘과 같은 양자역학 원리를 이용해 절대적으로 안전한 통신을 목표로 하는 차세대 네트워크 기술이 활발히 연구 개발되고 있습니다.
